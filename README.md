# FoodieBot 🍔🍕🤖

FoodieBot is a food ordering chatbot developed using Dialogflow ES. The chatbot allows users to browse food categories, select menu items, provide contact information, and complete the checkout process through a conversational interface.

## Project Overview

This project demonstrates the implementation of a context-based conversational chatbot using Dialogflow ES. The chatbot guides users through the food ordering process while maintaining conversation state using Dialogflow Contexts.

## Features

- Welcome and Greeting Intents
- Interactive Menu Navigation
- Category-Based Food Selection
- Food Ordering Workflow
- Email Collection
- Phone Number Collection
- Checkout Process
- Context-Based Conversation Management
- User-Friendly Conversational Interface

## Technologies Used

- Dialogflow ES
- Google Cloud Platform
- Intents
- Contexts
- Training Phrases
- Natural Language Understanding (NLU)

## Conversation Flow

```text
User Greeting
      ↓
  Show Menu
      ↓
Select Category
      ↓
Select Food Item
      ↓
  Enter Email
      ↓
Enter Phone Number
      ↓
   Checkout
      ↓
Order Confirmation
```

## Sample Conversation

```text
User: Hi

Bot: Welcome to FoodieBot! 🍔🍕

User: Show Menu

Bot: Displays available food categories

User: Burger

Bot: Displays burger menu

User: Chicken Burger

Bot: Please enter your Email ID

User: steff@gmail.com

Bot: Please enter your Contact Number

User: 9876543210

Bot: Type Checkout to place your order

User: Checkout

Bot: 🎉 Order Confirmed!
```
